HR Analytics: Employee Attrition Insights
Project Overview

This project analyzes employee attrition using Python, performing data cleaning, exploratory data analysis (EDA), and visualizations to uncover trends and insights. The analysis helps HR teams understand factors influencing employee turnover and provides actionable insights for retention strategies.

Dataset Source: IBM HR Analytics Attrition Dataset

Technologies Used

Python – Data analysis and scripting

Pandas & NumPy – Data cleaning and manipulation

Matplotlib & Seaborn – Static visualizations

Plotly – Interactive visualizations

Jupyter Notebook – Project development environment

Key Features

Performed data cleaning to handle missing values and duplicates

Conducted exploratory data analysis (EDA) to discover patterns

Generated visualizations for age, income, department, and attrition trends

Created interactive charts with Plotly for dynamic insights

Calculated correlations between employee features and attrition
